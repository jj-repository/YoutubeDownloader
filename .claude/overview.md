# Overview

v5.0.4 — Downloads video/audio from YouTube and supported sites.

## Features
- Quality selection with file size preview
- Thumbnail preview with caching
- Trimming (start/end sliders + preview frames)
- Clipboard monitoring for URLs (Clipboard Mode tab)
- Catbox upload with history (Uploader tab)
- Dark/light QSS theme, dark title bar on Windows (DWM/ctypes)

## Files
- `downloader_pyqt6.py` — main app (PyQt6, ~6000 lines)
- `downloader.py` — legacy tkinter, archived, ignore
- `constants.py` — all config constants
- `requirements.txt`

Untracked scratch files (ignore): `_port_callbacks.py`, `_port_download.py`, `_port_updates.py`, `downloader_pyqt6_part1.py`
